Commands

コマンド一覧

Linux/Unixコマンドの説明と使い方

コマンドを検索

apt
Debian系Linuxディストリビューションでパッケージの検索、インストール、更新、削除を行うパッケージ管理ツール。
apt-file
Debian系ディストリビューションでパッケージに含まれるファイルを検索するツール。
awk
テキストファイルを処理し、パターンマッチングに基づいて特定のアクションを実行するテキスト処理ツール。
basename
ファイルパスからディレクトリ部分を削除し、ファイル名のみを取得します。
cat
ファイルの内容を表示したり、複数のファイルを連結したりします。
chgrp
ファイルやディレクトリのグループ所有権を変更します。
chmod
ファイルやディレクトリのアクセス権限を変更します。
chsh
ユーザーのログインシェルを変更します。
cmp
ファイルを比較し、最初に異なるバイトを見つけます。
continue
シェルスクリプトで中断された繰り返し処理を再開します。
cp
ファイルやディレクトリをコピーします。
curl
URLで指定されたリソースをダウンロードまたは操作するためのコマンドラインツール。
cut
テキストファイルから特定の列や文字を抽出するためのコマンド。
date
現在の日付と時刻を表示または設定します。
dd
ファイルの変換とコピーを行うユーティリティ。
df
ファイルシステムのディスク容量使用状況を表示します。
diff
ファイル間の違いを比較して表示します。
diff3
3つのファイル間の違いを行ごとに比較して表示します。
dig
DNSサーバーに対してクエリを実行し、ドメイン名の情報を取得します。
dirname
パス名からディレクトリ部分を抽出します。
dmesg
カーネルのリングバッファからシステムメッセージを表示します。
docker
コンテナの作成、管理、実行を行うためのコマンドラインツール。
docker-build
Dockerfileからイメージを構築するためのコマンドです。
docker-compose
複数のコンテナを定義し、実行するためのツール。
docker-run
新しいコンテナを作成し、指定されたイメージから起動するコマンドです。
dpkg
Debian ベースのシステム（Ubuntu など）でパッケージの管理を行うコマンドラインツール。
du
ディスク使用量を表示するコマンドです。
echo
標準出力に文字列やテキストを表示します。
egrep
テキストファイル内でパターンを検索するための拡張正規表現を使用するコマンド。
env
現在の環境変数を表示または変更します。
eval
シェル内で文字列をコマンドとして評価・実行します。
expr
式を評価し、その結果を標準出力に表示します。
false
常に失敗を返す条件式として使用されるコマンド。
file
ファイルの種類を判別します。
find
ファイルシステム内でファイルやディレクトリを検索します。
for
指定した範囲の値や項目に対して繰り返し処理を実行します。
free
システムのメモリ使用状況を表示します。
gdb
プログラムのデバッグや実行時の動作を分析するためのコマンドラインデバッガです。
getent
データベースエントリを取得し表示するコマンド。
getopt
コマンドラインオプションを解析し、シェルスクリプト内でのオプション処理を簡素化します。
git
バージョン管理システムで、ファイルの変更履歴を追跡し、複数の開発者との共同作業を可能にします。
git-add
ワーキングディレクトリの変更をステージングエリア（インデックス）に追加します。
git-branch
ブランチの作成、一覧表示、名前変更、削除を行います。
git-commit
変更をリポジトリに記録します。
git-merge
複数のコミット履歴を統合します。
git-pull
リモートリポジトリから変更を取得し、ローカルブランチに統合します。
git-push
リモートリポジトリに変更を送信するコマンドです。
git-rebase
コミット履歴を再構成し、ブランチの起点を変更する。
git-reset
コミットポインタを特定のコミットに移動させ、作業ツリーやインデックスを選択的にリセットします。
git-stash
作業中の変更を一時的に保存し、クリーンな作業ディレクトリに戻すためのコマンド。
git-switch
ブランチを切り替えるコマンドです。
gpasswd
グループのパスワードとメンバーシップを管理します。
grep
テキストファイルやストリーム内のパターンを検索します。
groupadd
新しいグループをシステムに追加します。
groupdel
システムからグループを削除するコマンドです。
groupmod
既存のグループの属性を変更します。
groups
ユーザーが所属するグループを表示します。
gunzip
圧縮ファイルを展開するコマンドです。
gzip
ファイルを圧縮・展開するためのコマンドラインユーティリティです。
hd
ファイルの内容を16進数形式で表示します。
head
ファイルの先頭部分を表示します。
host
DNS（ドメインネームシステム）の検索を実行し、ドメイン名からIPアドレスへの変換や、その逆の変換を行います。
hostname
システムのホスト名を表示または設定します。
http
HTTP リクエストを送信し、レスポンスを表示するコマンドラインツールです。
install
ファイルをコピーし、モード、所有者、グループを設定するコマンドです。
iostat
システムの入出力（I/O）統計情報を表示します。
iotop
I/O 使用量をプロセスごとに監視するリアルタイムモニタリングツール。
ip
ネットワークインターフェース、ルーティング、トンネルなどのネットワーク設定を表示・管理します。
journalctl
システムログを表示・管理するためのコマンドです。
jq
JSONデータを柔軟に処理・変換するためのコマンドラインツール。
kill
プロセスを終了させるためのシグナルを送信します。
killall
プロセス名を指定して、該当するすべてのプロセスを終了させる。
[
条件式を評価するためのテストコマンドです。
less
テキストファイルを画面単位で閲覧するためのページャー。
ln
ファイルやディレクトリへのリンク（ショートカット）を作成します。
ls
ファイルやディレクトリの情報を表示します。
lsof
システム上で開いているファイルを一覧表示します。
ltrace
実行中のプログラムが行うライブラリコール（関数呼び出し）をトレースします。
mkfifo
名前付きパイプ（FIFO特殊ファイル）を作成します。
more
テキストファイルを画面単位で表示するページャー。
mv
ファイルやディレクトリを移動（名前変更）するためのコマンド。
nc
ネットワーク接続を読み書きするためのユーティリティ。
od
ファイルやデータをさまざまな形式（8進数、16進数など）でダンプ表示します。
openssl
暗号化、証明書管理、ハッシュ計算などの暗号操作を実行するためのツールです。
pandoc
ドキュメント形式を変換するための汎用的なコンバーター。
passwd
ユーザーアカウントのパスワードを変更します。
patch
ファイルに差分（パッチ）を適用するユーティリティ。
perf
システムのパフォーマンスを分析し、プロファイリングするためのツール。
ping
ネットワーク上のホストに対して接続性をテストし、応答時間を測定します。
printf
指定されたフォーマットに従ってテキストを整形して出力します。
ps
実行中のプロセスに関する情報を表示します。
pstack
実行中のプロセスのスタックトレースを表示します。
pstree
プロセスツリーを表示します。
pwd
現在の作業ディレクトリのフルパスを表示します。
readlink
シンボリックリンクの参照先（リンク先）を表示します。
realpath
ファイルやディレクトリの絶対パス（正規化されたフルパス）を表示します。
rg
ファイル内の文字列を高速に検索するためのツール。
rm
ファイルやディレクトリを削除します。
rmdir
ディレクトリを削除します。
rsync
ファイルやディレクトリを効率的に同期・コピーするためのツール。
scp
ネットワーク経由でファイルやディレクトリを安全にコピーします。
script
ターミナルセッションの全入出力を記録し、タイプスクリプトファイルに保存します。
sed
テキストストリームを変換・編集するためのストリームエディタです。
seq
指定された範囲の数値を順番に出力します。
set
シェル変数の設定や表示、シェルの動作を制御するためのコマンドです。
sftp
セキュアなファイル転送プロトコル（SSH File Transfer Protocol）を使用してファイルを転送します。
shift
シェルスクリプト内で位置パラメータをシフトします。
sleep
指定した時間だけプロセスを一時停止（スリープ）させます。
sort
テキストファイルの行を並べ替えます。
source
シェルスクリプトやコマンドを現在のシェル環境で実行します。
split
ファイルを複数の小さなファイルに分割します。
ss
ネットワークソケット情報を表示するユーティリティです。
ssh
リモートホストに安全に接続し、ログインするためのコマンド。
ssh-add
SSH 認証エージェントに秘密鍵を追加・管理するコマンドです。
ssh-agent
SSH認証鍵を安全に管理し、パスフレーズの再入力なしで認証を行うためのエージェントを起動します。
ssh-copy-id
リモートホストに SSH 公開鍵を安全にコピーするためのユーティリティです。
ssh-keygen
SSH認証に使用する公開鍵と秘密鍵のペアを生成するツール。
stat
ファイルやファイルシステムの詳細な情報を表示します。
strace
プロセスのシステムコールとシグナルをトレースするためのユーティリティ。
su
ユーザーを切り替えるコマンドです。
sudo
他のユーザー（通常はroot）の権限でコマンドを実行します。
sudoedit
特権が必要なファイルを安全に編集するためのコマンド。
systemctl
システムとサービスマネージャーを制御するコマンド。
tail
ファイルの末尾部分を表示します。
tar
ファイルやディレクトリをアーカイブ（まとめて一つのファイルに）したり、圧縮・展開したりします。
tcpdump
ネットワークパケットをキャプチャして分析するためのコマンドラインツール。
tee
標準入力からデータを読み取り、標準出力とファイルの両方に書き込みます。
top
システムのプロセスをリアルタイムで監視し、CPU・メモリ使用率などのリソース情報を表示します。
touch
ファイルのタイムスタンプを更新したり、新しい空のファイルを作成したりします。
tr
文字の変換や削除を行います。
tree
ディレクトリ構造をツリー形式で表示します。
true
何も実行せずに常に成功（終了コード 0）を返す。
truncate
ファイルのサイズを指定したサイズに変更します。
uname
システムの情報を表示するコマンドです。
uniq
テキストファイル内の重複行を検出または削除します。
unlink
ファイルへのリンクを削除します。
until
指定した条件が真になるまでコマンドを繰り返し実行するシェルの制御構造です。
unzip
ZIP アーカイブファイルを展開するコマンド。
update-alternatives
システム上の複数のバージョンのコマンドやライブラリを管理し、デフォルトを設定するためのツールです。
update-locale
システムのロケール設定を表示または変更するコマンド。
visudo
sudoers ファイルを安全に編集するためのコマンド。
vmstat
システムの仮想メモリ統計情報を表示します。
w
現在ログインしているユーザーとその活動状況を表示します。
wc
テキストファイルの行数、単語数、バイト数をカウントします。
wget
リモートサーバーからファイルをダウンロードするためのコマンドラインツールです。
which
指定したコマンドの実行ファイルのパスを表示します。
while
条件が真である間、コマンドを繰り返し実行します。
whoami
現在のユーザー名を表示します。
xargs
標準入力からコマンドラインを構築し、他のコマンドを実行します。
xdg-mime
ファイルタイプとそれに関連付けられたアプリケーションを管理するコマンド。
xdg-open
指定したファイルやURLをデフォルトのアプリケーションで開きます。
yes
指定された文字列を繰り返し出力します。
zip
ファイルを圧縮して ZIP アーカイブを作成します。
