noman's simplified page

ls command

The ls command lists files in a directory.

Options:
  -l (long format)
    Shows detailed information about files
    
  -a (all)
    Shows hidden files (starting with .)
    
  -h (human-readable)
    Shows file sizes in KB, MB, GB
    
Example:
  $ ls -lha
  total 24K
  drwxr-xr-x 4 user staff 128B Apr 10 15:35 .
  -rw-r--r-- 1 user staff 1.0K Apr 10 15:30 document.txt
  -rw-r--r-- 1 user staff  74B Apr 10 15:32 .hidden
...

pip install noman
