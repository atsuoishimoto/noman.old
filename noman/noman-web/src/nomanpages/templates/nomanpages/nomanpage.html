grep

A powerful pattern matching tool for text filtering.

Usage

Basic usage:

grep [OPTIONS] PATTERN [FILE...]

Example:

grep -i "error" /var/log/syslog

Common Options

-i Case-insensitive search
-r Recursive search in subdirectories
-n Show line numbers
--color Highlight matched patterns

Examples

Search for the word "ERROR" in all log files, ignoring case:

grep -i "error" /var/log/*.log

FAQ / Tips

Q: How to invert match?

Use the -v option. Example: grep -v "pattern" file

Q: How do I search multiple files?

Simply list them after the pattern: grep "TODO" file1 file2 file3
