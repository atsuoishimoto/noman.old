コマンド一覧

Linux/Unix コマンドの解説

コマンドを検索

apt
Debian系Linuxディストリビューション（Ubuntuなど）用のパッケージ管理ユーティリティです。
apt-file
APTパッケージ管理システム内のパッケージ内のファイルを検索します。
awk
テキストファイルのパターンスキャンと処理を行う言語です。
basename
パス名からファイル名やディレクトリ名を抽出します。
cat
ファイルの内容を表示したり、複数のファイルを連結したりします。
chgrp
ファイルやディレクトリのグループ所有権を変更します。
chmod
ファイルやディレクトリのモードビット（アクセス権限）を変更します。
chsh
ユーザーのログインシェルを変更します。
cmp
2つのファイルをバイト単位で比較します。
continue
一時停止したジョブをフォアグラウンドまたはバックグラウンドで再開します。
cp
ファイルやディレクトリをソースから宛先へコピーします。
curl
URLを使用してサーバーとの間でデータを転送するツールで、HTTP、HTTPS、FTP、SFTPなど多数のプロトコルをサポートします。
cut
各ファイルの行から選択した部分を抽出し、標準出力に出力します。
date
システムの日付と時刻を表示または設定します。
dd
ファイルのブロックレベル操作によるコピーと変換を行います。
df
マウントされたファイルシステムのディスク容量使用状況を表示します。
diff
ファイルを行ごとに比較します。
diff3
3つのファイルを行ごとに比較し、差分を表示します。
dig
DNSネームサーバーにドメイン情報を問い合わせます。
dirname
パス名からディレクトリ部分を出力します。
dmesg
カーネルメッセージバッファを表示または制御します。
docker
Dockerコンテナ、イメージ、ネットワーク、その他のDockerオブジェクトを管理します。
docker-build
Dockerfileからイメージをビルドします。
docker-compose
複数のDockerコンテナを定義して実行するためのツールです。
docker-run
指定したイメージから新しいコンテナを作成して起動します。
dpkg
Debian ベースのシステム（Ubuntu など）で Debian パッケージファイル（.deb）を管理する。
du
ファイルやディレクトリの使用ディスク容量を推定します。
echo
テキストや変数を標準出力に表示します。
egrep
テキスト内で拡張正規表現を使用してパターンを検索します。
env
現在の環境変数を表示したり、変更された環境で指定したコマンドを実行したりします。
eval
引数からコマンドを構築して実行します。
expr
式を評価し、結果を出力します。
false
引数に関係なく失敗ステータスを返します。
file
ファイルの内容を調べてファイルタイプを判定します。
find
ディレクトリ階層内でファイルを検索します。
for
リスト内の各項目に対してコマンドまたはコマンドセットを実行します。
free
システム内の空きメモリと使用中メモリの量を表示します。
gdb
GNU デバッガを使用してソースコードレベルでプログラムをデバッグおよび分析します。
getent
管理データベース（ホスト、ユーザー、グループなど）からエントリを取得します。
getopt
コマンドラインオプションを標準化された方法で解析します。
git
分散型バージョン管理システムで、ソフトウェア開発中のソースコードの変更を追跡します。
git-add
ファイルの内容をステージングエリア（インデックス）に追加し、次のコミットの準備をします。
git-branch
Gitリポジトリ内のブランチを一覧表示、作成、または削除します。
git-commit
リポジトリに変更を記録し、インデックスの現在の内容を含む新しいコミットを作成します。
git-merge
複数の開発履歴を統合します。
git-pull
リモートリポジトリやローカルブランチから変更を取得して統合します。
git-push
リモートリポジトリに対してローカルの参照と関連オブジェクトを更新します。
git-rebase
別のベースの先端に対してコミットを再適用します。
git-reset
指定した状態に現在のHEADをリセットします。
git-stash
作業ディレクトリの変更を一時的に保存して後で使用するためのコマンドです。
git-switch
ブランチを切り替えたり、作業ツリーのファイルを復元したりします。
gpasswd
グループのメンバーシップと認証情報を /etc/group および /etc/gshadow ファイルで管理します。
grep
ファイル内のパターンを検索します。
groupadd
システムに新しいグループを作成します。
groupdel
システムからグループを削除します。
groupmod
システム上のグループ定義を変更します。
groups
ユーザーが所属するグループを表示します。
gunzip
gzipで圧縮されたファイルを展開します。
gzip
ファイルをLempel-Ziv符号化（LZ77）を使用して圧縮または展開します。
hd
ファイルの内容を16進数、10進数、8進数、またはASCII形式で表示します。
head
ファイルの先頭部分を表示します。
host
ドメインネームサーバーに対してDNS検索を行うユーティリティです。
hostname
システムのホスト名を表示または設定します。
http
コマンドラインからHTTPリクエストを送信し、レスポンスを表示します。
install
ファイルをコピーして属性を設定します。
iostat
CPUおよびデバイスとパーティションの入出力統計情報を表示します。
iotop
システム上のプロセスによるI/O使用状況を監視します。
ip
Linuxシステム上のネットワークインターフェース、ルーティング、トンネルを表示および操作します。
ipcs
アクティブなIPCファシリティ（共有メモリ、メッセージキュー、セマフォ）に関する情報を表示します。
journalctl
systemdジャーナルからメッセージを検索して表示します。
jq
JSONデータを軽量で柔軟なコマンドラインプロセッサで処理・変換します。
kill
プロセスにシグナルを送信し、通常はプロセスを終了させます。
killall
名前を指定してプロセスを終了させるコマンド。
[
条件式を評価し、評価結果に基づいてステータスを返します。
less
テキストファイルをスクロール可能なインターフェースで表示し、前後の移動を可能にします。
ln
ファイル間のリンクを作成します。
ls
ディレクトリの内容を一覧表示します。
lsof
システム上で開いているファイルとそれを開いているプロセスを一覧表示します。
ltrace
プログラムのライブラリ呼び出しをトレースします。
mkdir
ディレクトリ（フォルダ）を作成します。
mkfifo
指定した名前の名前付きパイプ（FIFO）を作成します。
more
ファイルの内容を1画面ずつ表示します。
mv
ファイルやディレクトリを移動または名前変更します。
nc
ネットワーク接続を作成・管理し、データ転送、ポートスキャン、ネットワークデバッグを行います。
od
ファイルの内容を様々な形式（主に8進数、16進数、その他の数値表現）で表示します。
openssl
暗号化通信、証明書管理、および様々な暗号操作のための暗号機能を提供します。
pandoc
様々な形式の文書を相互に変換します。
passwd
ユーザーパスワードを変更します。
patch
差分ファイルを元のファイルに適用します。
perf
Linuxのパフォーマンス分析ツールで、ハードウェアカウンタ統計やトレース機能を提供します。
ping
ネットワークホストに ICMP ECHO_REQUEST パケットを送信して接続性を確認します。
printf
指定されたフォーマット文字列に従ってデータを整形し、出力します。
ps
実行中のプロセスに関する情報を表示します。
pstack
実行中のプロセスのスタックトレースを表示します。
pstree
プロセスの親子関係を示すツリーを表示します。
pwd
現在の作業ディレクトリの絶対パスを表示します。
readlink
シンボリックリンクの解決先またはファイルの正規名を表示します。
realpath
ファイルの解決された絶対パスを表示します。
rg
正規表現を使用してファイル内のパターンを検索し、ディレクトリを再帰的に探索します。
rm
ファイルやディレクトリをファイルシステムから削除します。
rmdir
ファイルシステムから空のディレクトリを削除します。
rsync
ファイルやディレクトリをローカルまたはリモート間で同期します。
scp
ネットワーク上のホスト間でSSHを使用してファイルを安全にコピーします。
script
ターミナルセッションの全アクティビティを記録し、タイプスクリプトを作成します。
sed
テキストのフィルタリングと変換のためのストリームエディタ。
seq
指定した範囲の数値シーケンス（連続した数字）を出力します。
set
シェルオプションと位置パラメータを表示または設定します。
sftp
暗号化された接続を介してホスト間でファイルを安全に転送します。
shift
シェルスクリプト内の位置パラメータをシフトし、最初のパラメータを削除して他のすべてのパラメータを1つ下の位置に移動します。
sleep
指定した時間だけ実行を一時停止します。
sort
テキストファイルの行をアルファベット順または数値順に並べ替えます。
source
現在のシェル環境内でファイルやスクリプトからコマンドを実行します。
split
ファイルを複数の小さな部分に分割します。
ss
ソケット統計を表示し、ネットワーク接続に関する情報を提供します。
ssh
リモートマシンに暗号化されたネットワーク接続を介して安全に接続します。
ssh-add
SSH認証エージェントに秘密鍵を追加し、接続認証に使用します。
ssh-agent
SSH秘密鍵を管理する認証エージェントで、接続時にパスフレーズの再入力が不要になります。
ssh-copy-id
リモートマシンの authorized keys ファイルに公開鍵をインストールします。
ssh-keygen
SSH認証用の鍵ペアを生成、管理、変換します。
stat
ファイルやファイルシステムの詳細な状態情報を表示します。
strace
プロセスが行うシステムコールとシグナルをトレースします。
su
ユーザーIDを切り替えたり、一時的に別のユーザーになったりします。
sudo
別のユーザー（通常は管理者権限を持つユーザー）として、コマンドを実行します。
sudoedit
特権ユーザーとしてファイルを安全に編集し、ファイルの所有権と権限を維持します。
systemctl
システムとサービスマネージャーである systemd を制御します。
tail
ファイルの末尾部分を表示します。
tar
ファイルをアーカイブファイルとして作成、抽出、または内容を一覧表示するためのコマンド。
tcpdump
ネットワークトラフィックをキャプチャして分析するツール。
tee
標準入力から読み取り、標準出力とファイルの両方に同時に書き込みます。
top
プロセスに関する情報をリアルタイムで表示・更新します。
touch
ファイルのタイムスタンプを作成または更新します。
tr
標準入力から文字を変換または削除し、標準出力に書き込みます。
tree
ディレクトリの内容をツリー状のフォーマットで表示し、ディレクトリとファイルの階層構造を視覚化します。
true
常に成功の終了ステータス（0）を返します。
truncate
ファイルのサイズを指定したサイズに縮小または拡張します。
uname
システムのオペレーティングシステムに関する情報を表示します。
uniq
隣接する重複行を入力からフィルタリングまたは報告します。
unlink
ファイルシステムから単一のファイルを削除します。
until
条件が満たされるまでコマンドを繰り返し実行します。
unzip
ZIPアーカイブからファイルを展開します。
update-alternatives
代替システム内のシンボリックリンクを管理し、デフォルトコマンドを決定します。
update-locale
システム全体のロケール設定を /etc/default/locale ファイルを更新することで設定します。
visudo
sudoers ファイルを構文チェック付きで安全に編集します。
vmstat
仮想メモリの統計情報を表示し、システムのプロセス、メモリ、ページング、ブロックI/O、トラップ、およびCPUアクティビティに関する情報を提供します。
w
現在ログインしているユーザーと彼らの行動を表示します。
wc
ファイル内の行数、単語数、バイト数をカウントします。
wget
ウェブからHTTP、HTTPS、またはFTPプロトコルを使用してファイルをダウンロードします。
which
PATH環境変数を検索してコマンドの場所を特定します。
while
条件が真である限り、コマンドを繰り返し実行します。
whoami
現在の実効ユーザー名を表示します。
xargs
標準入力から引数を受け取り、コマンドを実行します。
xdg-mime
デスクトップ環境でファイルタイプの関連付けを照会・設定します。
xdg-open
ファイルやURLをユーザーの優先アプリケーションで開きます。
yes
文字列を中断されるまで繰り返し出力します。
zip
ファイルを圧縮アーカイブにパッケージ化します。
